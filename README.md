# Kafu

A WebAssembly-based framework for building and deploying distributed applications across edge and cloud environments.

## Installation

### Linux

```sh
curl -fsSL "https://raw.githubusercontent.com/tamaroning/kafu/refs/heads/main/install.sh" | sh
```

### macOS

This platform is not supported.

### Windows

This platform is not supported.
